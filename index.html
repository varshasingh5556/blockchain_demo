🔗 Blockchain-Based Order Tracking System

📌 About This Project

This project demonstrates how Blockchain technology ensures data integrity, transparency, and security.

Each step of an order is stored as a block and linked using cryptographic hashes, forming a secure chain.

Smart contract logic ensures steps follow the correct sequence and cannot be skipped.

🧠 How It Works

Each block stores its own data and the hash of the previous block. Any change in data alters the hash and breaks the chain.

Mining (Proof of Work) increases the difficulty of recomputing hashes, making attacks computationally expensive.

In a multi-node system, all nodes maintain a copy of the blockchain and synchronize with each other.
